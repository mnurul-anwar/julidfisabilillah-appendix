Metodologi

Dataset & Pendekatan Analisis

Penelitian ini menggunakan pendekatan mixed methods yang menggabungkan analisis kuantitatif berbasis Python dan analisis kualitatif interpretatif. Berikut ringkasan tiga pendekatan analitis yang digunakan untuk masing-masing dimensi.

01

Struktur Peluang

Analisis runtun waktu (time series) dengan deteksi puncak berbasis Z-score dan 7-day rolling mean. Hari dikategorikan sebagai puncak apabila Z-score ≥ 2,0.

02

Struktur Mobilisasi

Pengelompokan partisipan ke dalam enam tier groups berdasarkan frekuensi cuitan, dikombinasikan dengan cohort analysis berdasarkan bulan pertama partisipasi.

03

Proses Pembingkaian

Klasifikasi berbasis aturan (rule-based classification) menggunakan codebook operasional tiga kategori utama: diagnostic, prognostic, dan motivational framing.

# ── Struktur Peluang: Z-score & Rolling Mean ──────────────────────────
import pandas as pd
import numpy as np

daily = df.groupby('date_only').size().reset_index(name='tweet_count')

# Lengkapi tanggal yang kosong dengan 0
full_range = pd.date_range(daily['date_only'].min(), daily['date_only'].max())
daily = daily.set_index('date_only').reindex(full_range, fill_value=0).reset_index()

# Rolling mean 7 hari (terpusat)
daily['rolling_mean'] = daily['tweet_count'].rolling(7, center=True, min_periods=1).mean()

# Z-score harian
mu  = daily['tweet_count'].mean()   # μ = 19.52
sig = daily['tweet_count'].std()    # σ = 25.87
daily['zscore'] = (daily['tweet_count'] - mu) / sig

# Hari puncak: Z-score ≥ 2.0
peaks = daily[daily['zscore'] >= 2.0]

# ── Struktur Mobilisasi: Tier Groups ─────────────────────────────────
user_counts = df.groupby('username').size().reset_index(name='tweet_count')

tier_bins   = [0, 1, 10, 25, 50, 250, 500]
tier_labels = ['Tier 6', 'Tier 5', 'Tier 4', 'Tier 3', 'Tier 2', 'Tier 1']
user_counts['tier'] = pd.cut(user_counts['tweet_count'], bins=tier_bins, labels=tier_labels)

# ── Pembingkaian: Rule-based Classification ───────────────────────────
diagnostic_kw  = ['zionis', 'israel', 'penjajah', 'genosida', 'zionis pesek', ...]
prognostic_kw  = ['ayo', 'report', 'boikot', 'rapatkan barisan', 'serangan', ...]
motivational_kw = ['bismillah', 'semangat', 'fi sabilillah', 'insyaallah', ...]

def classify_frame(text):
    d = any(kw in text for kw in diagnostic_kw)
    p = any(kw in text for kw in prognostic_kw)
    m = any(kw in text for kw in motivational_kw)
    return {'diagnostic': d, 'prognostic': p, 'motivational': m}

df['framing'] = df['text_clean'].apply(classify_frame)

Dimensi 3

Proses Pembingkaian (Framing Processes)

Klasifikasi framing dilakukan dengan pendekatan berbasis aturan (rule-based) mengikuti tiga tugas pembingkaian inti Snow dkk. (1986): diagnostic, prognostic, dan motivational framing. Satu tweet dapat memperoleh lebih dari satu label.

Distribusi Kategori

Distribusi Framing (Donut)
Diagnostic Framing

Mengidentifikasi masalah dan mengatribusikan kesalahan kepada aktor tertentu. Pola dominan: konstruksi Israel sebagai pelanggar kemanusiaan, "zionis pesek" sebagai musuh internal.

Rata-rata engagement: 886,45
Prognostic Framing

Menawarkan solusi, strategi, dan bentuk tindakan. Meski proporsinya paling kecil, memiliki rata-rata engagement tertinggi. Terpusat pada akun komando @erlanishere.

Rata-rata engagement: 1.562,8
Motivational Framing

Memberikan legitimasi moral dan dorongan partisipasi. Menggabungkan diksi keagamaan dan nasionalisme historis sebagai kosakata motif yang mempertahankan kontinuitas gerakan.

Rata-rata engagement: 143,56
